# myOpic - Releases

Binary releases for the myOpic optical spectrometer application.

## Downloads

See the [Releases](https://github.com/nhatminh2h/Myopic-releases/releases) page for downloads.

## Available Builds

| Platform | Architecture | File |
|----------|--------------|------|
| Linux (Raspberry Pi) | ARM64 (aarch64) | `Myopic_arm64` |

## Usage

```bash
chmod +x Myopic_arm64
./Myopic_arm64
```

Requires a display (HDMI monitor or preferably X11 forwarding).
