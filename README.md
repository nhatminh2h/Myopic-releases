# Cortica PhovIR External - Releases

Binary releases for the Cortica PhovIR External application.

## Downloads

See the [Releases](https://github.com/nhatminh2h/NottinghamUniCCP-releases/releases) page for downloads.

## Available Builds

| Platform | Architecture | File |
|----------|--------------|------|
| Linux (Raspberry Pi) | ARM64 (aarch64) | `Cortica_PhovIR_External_arm64` |

## Usage

```bash
chmod +x Cortica_PhovIR_External_arm64
./Cortica_PhovIR_External_arm64
```

Requires a display (HDMI monitor or X11 forwarding).
